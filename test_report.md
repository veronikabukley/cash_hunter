## Предложения по улучшению

### 1. Добавить фильтр по тендеру в Retriever

```python
def _filter_tender(self, df: pd.DataFrame, has_tender: Optional[bool]) -> pd.DataFrame:
    if has_tender is True:
        df = df[df["Наличие_тендера"].str.lower() == "да"]
    elif has_tender is False:
        df = df[df["Наличие_тендера"].str.lower() != "да"]
    return df
```

Параметр `has_tender: Optional[bool]` в `search()` — `True` = только с тендером, `False` = только без, `None` = все.

### 2. Добавить метод `search_all()` в Retriever

```python
def search_all(self, sphere=None, min_revenue=0, min_cash=0, has_tender=None):
    df = self._load()
    df = self._filter_sphere(df, sphere)
    df = self._filter_min_revenue(df, min_revenue)
    df = self._filter_min_cash(df, min_cash)
    df = self._filter_tender(df, has_tender)
    return self._sort(df)
```

Сделать параметр `region` опциональным в `search()` или добавить отдельный метод.

### 3. Улучшить генерацию скриптов

- Использовать **комбинированный шаблон** всегда (название + регион + сфера + выручка + наличные + тендер), а не выбирать один из вариантов случайно
- Или генерировать скрипт через LLM (OpenAI/ZAI) с подстановкой всех данных компании

### 4. Добавить ценовые колонки в данные

В `generate_data.py` добавить колонки:
- `Цена_за_1` — тариф для 1 посещения/месяца
- `Цена_за_5` — тариф для 5 посещений
- `Цена_за_25` — тариф для 25 посещений
- `Цена_за_50` — тариф для 50+ посещений
- Опционально: `Цена_премиум` — премиум-тариф с доп. услугами

### 5. Добавить аналитические методы в агента

- `average_revenue(sphere)` — средняя выручка по сфере
- `compare_regions(region1, region2, metric)` — сравнение регионов по метрике
- `top_n_by_cash(sphere, has_tender, n)` — топ-N с фильтрами

### 6. Устранить случайность в выборе шаблона

`random.choice` в `generate_scripts` создаёт неконсистентность — при каждом запуске скрипты разные. Для прода нужен детерминированный выбор шаблона по логике (например: всегда комбинированный шаблон, или выбор по соотношению наличные/тендер).
